Class: BinderManager

BinderManager

Creates an instance of the binder management class

new BinderManager(baseDirectory)

Creates a new instance of the BinderManager class. An instance is used to manage multiple binder instances and centralize their configuration details.

The baseDirectory option points to a directory that will hold another sub directory named "binders". Each of these directories hold the configuration details for a single Binder.

Parameters:
Name Type Description
baseDirectory string

the location where all of the notebook configurations that are managed by this instance are stored.
